1、创建一个项目npm iniy --yes
2、安装typescript, npm i typescript -D
3、创建文件夹src，并且在src下面创建1.ts文件，写点js的东西。然后编译它：`.\node_modules\.bin\tsc .\src\1.ts`






### 配置文件
|- tsconfig.json   配置文件
常见的配置如下：
```
{
    "compilerOptions": {
        "module": "system",  // 使用的模块化系统
        "outDir": "./dist",  // 编译之后的文件存储位置
        "outFile": "./bundle.js"   // 输出合并成一个文件
    }
}
```
